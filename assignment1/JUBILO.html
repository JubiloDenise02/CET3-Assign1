Home

The home page is the first Web page that is displayed after starting a Web browser like Netscape's Navigator or Microsoft's Internet Explorer. The browser is usually preset so that the home page is the first page of the browser manufacturer.

Profile

A profile page represents information regarding a user's identity on a website or in a mobile app. It may contain personal data, a profile photo, summary, interests, achievements, and more.

Settings

Website settings is a single place where various global and system-level content settings can be configured for your website's blog, navigation, pages, and themes.
